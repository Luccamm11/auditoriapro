## üîç SEO Analysis Checklist

### 1. **Technical SEO**
Evaluate:
- HTML semantic structure
- Correct usage of `<title>` and `<meta description>`
- Meta robots configuration
- Heading hierarchy (H1‚ÄìH6)
- Canonical tags
- Sitemap.xml presence and validity
- Robots.txt configuration
- Indexation signals
- URL structure and readability
- HTTPS and security
- Mobile-first readiness
- Core Web Vitals (LCP, CLS, INP/FID)
- Image optimization and lazy loading
- Error handling (4xx / 5xx)

---

### 2. **On-Page SEO**
Evaluate:
- Keyword usage and placement
- Content relevance to target keywords
- Duplicate content signals
- Internal linking structure
- Anchor text quality
- Alt text usage on images

---

### 3. **Content SEO**
Evaluate:
- Clarity of value proposition
- Search intent alignment
- Content depth and usefulness
- Scannability (headings, lists, sections)
- Content freshness and completeness

---

### 4. **Performance SEO**
Evaluate:
- Page load speed
- Asset size and optimization
- Render-blocking resources
- Cache usage
- CDN usage
- File minification

---

### 5. **UX & SEO Signals**
Evaluate:
- Navigation clarity
- Visual hierarchy
- Readability
- Accessibility basics
- User friction points
- CTA clarity and positioning

---

## üìä Report Structure (MANDATORY)

### 1. **Executive Summary**
- Overall SEO level: **Excellent / Good / Fair / Critical**
- Main strengths
- Main weaknesses

---

### 2. **Implemented Best Practices**
List clearly what the project already does well from an SEO standpoint.

---

### 3. **Issues Identified**
List issues and classify each as:
- **Critical**
- **Medium**
- **Low**

Explain:
- What is wrong
- Why it impacts SEO

---

### 4. **Improvement Opportunities**
For each opportunity:
- What should be improved
- Why it matters for SEO
- How to implement (technical or conceptual guidance)

---

### 5. **Actionable Checklist**
Provide a clear execution checklist, for example:
- [ ] Optimize meta titles
- [ ] Add or fix meta descriptions
- [ ] Implement sitemap.xml
- [ ] Improve LCP score
- [ ] Optimize images

---

### 6. **Organic Growth Potential**
Estimate the SEO growth potential **after applying the recommendations**, based on the current state of the project.
